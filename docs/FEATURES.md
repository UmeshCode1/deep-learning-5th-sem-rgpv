# Project Features

A quick tour of the capabilities provided by this repository and UI.

## UI (React + Vite + Tailwind)

- Modern, responsive design with tasteful 3D transforms and gradient accents
- Light/Dark theme toggle stored in localStorage
- Clean navigation: Syllabus, Practicals, Assignments
- Build-time optimized (Vite), ready for GitHub Pages deployment
- SPA-friendly with `404.html` fallback
